Ajax

什么是ajax?

  • Asynchronous javascript and xml 异步的javascript和xml

    Ajax不是一个技术,是多个技术联合实现的产物

  • Ajax是浏览器客户端上的前端技术,只要是做web开发, B/S架构的,不管服务器段编程语言哪一种,前端ajax都是要学习的

  • 异步和同步的区别

    a线程和B线程,并发执行,谁也不等谁,这就是异步

    a线程和b线程,a线程执行的时候b线程需要等待,这就是同步

  • 传统的请求和ajax请求的区别

    传统的请求,都是同步的

 Ajax请求,可以做到异步

  • ajax经典案例

    • 谷歌的地图
    • 输入框自动 补全
  • 浏览器本身这个软件也是支持多线程并发的,其中ajax请求就是一个线程,一个页面上可以同时发送多个ajax请求,多个ajax请求对应浏览器多个线程,当浏览器采用的是传统请求的时候,请求只要一发送,整个浏览器窗口锁定,无法点击其他按钮,并且浏览器会将窗口当中的数据全部清除,迎接新的页面
  • ajax主要解决的问题
    • 页面的局部刷新问题
    • 使用ajax可以在同一个网页中并发的发送多个请求,请求也请求直接互不干扰,这样可以提高用户的体验
