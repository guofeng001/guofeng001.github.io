Redis入门

Redis

1. 什么是reids

Redis是Remote Dictionary Server(远程数据服务)的缩写,**是意大利人开发的一款内存高速缓存数据库,它的数据模型为 key -value,支持丰富的数据结构(类型),可持久化,保证了数据安全**

简介

  1. 数据格式key:value
  2. 读写熟读非常快,常用于程序缓存

操作数据

redis是key-value的数据结构,每一条数据都是一个建值对,,建的类型是字符串,建不能重复,值得类型分为5种”:

  • string类型
  • hash哈希
  • list 列表(存储的数据类型是字符串,不可以有其他)
  • set 集合,(不可重复)
  • zset有序集合
